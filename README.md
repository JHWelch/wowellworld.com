# wowellworld.com

Homepage for [wowellworld.com](https://wowellworld.com) and various projects.

## Development

```sh
pnpm install
```

```sh
pnpm run dev
```

### Linting & Fixing

```sh
pnpm run lint
```

```sh
pnpm run fix
```

### Building

This app is built to GitHub Pages on merges to `main`.

To manually test a build locally to `dist`

```sh
pnpm run build
```
